What is the difference between RAM and ROM in computer systems?
🧠 DigitalTVET AI — from your module materials RAM (Random Access Memory) is temporary storage used while the computer is running — it loses data when powered off. ROM (Read-Only Memory) is permanent storage that retains data even without power, typically used to store firmware. Based on your Module 3 notes, RAM speed directly affects processing performance while ROM ensures the boot process runs correctly every time.
